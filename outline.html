Course Outline

Course: COMP 1950

Credits: 5

Length: 34 Lecture Hours

Cost: $350.00

"Prerequisites: " COMP 1850 " or Equivalent CSS & HTML Knowledge"

Course Information

"This hands-on course continues on from COMP 1850 Applied Web Development.
Students who have prior knowledge of HTML and CSS will explore web development in more depth with new examples and advanced techniques. "

"Participants will gain a deeper understanding of the latest technologies including HTML5 and CSS3 for developing web sites and applications.
There is a focus on responsive design and mobile presentation through the use of Media Queries. Adding behavior to web sites using JavaScript frameworks such as jQuery is introduced. "

"Labs and exercises focus on industry standard web development methodology.
Students gain experience creating templates for content management systems and presentation."

"By the end of this course successful students will be able to build dynamic and semantically marked-up content that is viewable on mobile and desktop screen sizes."

Course Goals

  • "To gain a deeper understanding of HTML, CSS and advanced topics in web development."
  • "To focus on the best practices in web page and web site design."

Learning Outcomes

After completing the course, students will have learned

  • To focus on the best practices in web page and web site design.
  • Create content using Javascript frameworks
  • Building websites for maximum compatibility
  • Understanding basic HTML/CSS theory and methodology.
  • Create professional looking typography using web-based fonts.
  • Understanding GIT and it's source control
